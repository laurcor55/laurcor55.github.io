Precise Enumeration with Flow Cytometry

Background

We wanted to see if an E. coli cell could complete division without a fully assembled Z ring. To do this, I used a temperature sensitive E. coli strain, JFL101. At 30 degrees, this strain behaves wild type. At 42 degrees, the Z ring disassembles. This disassembly happens within a minute and can be triggered with a rapid temperature shift.

Experimental Setup

For this experiment, I grew cells at 30 degrees. I then diluted the cells into a 42 degree water bath. I immediately took out an aliquot and fixed the cells in a formaldehyde/glutaraldehyde solution. This almost instantly halts growth and preserves the outer membrane. I repeated this fixation over time to obtain a pseudo time-course. I then counted the cells in fixed samples to create a growth curve over time.

Optimization of Temperature Jump Protocol

Finding Appropriate Software

Duke provides access to flow cytometry analysis software, but we only have a few copies. I didn't want to be tied to an on campus computer lab to do my analysis, so I sought an open source flow cytometry analysis package. After trying several, I found FlowCytometryTools by Jonathan Friedman and Eugene Yurtsev. I wrote a custom Python script to automatically read and gate FCS files. From this, I created a CSV file that I used for analyzing the data in R.

Calculating the Proper Cell Density

I counted cell fixed cells with a flow cytometer. Duke Cancer Institute has several BD FACSCanto II systems. These maintain quite a consistent flow rate, so they can be used to precisely count cells. I counted cells for a set amount of time, usually 1 to 3 minutes, and compared the relative increase in count over time.

To succeed in this experiment, I needed to determine the proper density of cells to produce an accurate cell count. If I counted cells that were not dense enough, the system's noise would overwhelm the true count. If I counted the cells at a density that was too high, the cells would overlap while going through the flow cytometer and would be under counted.

To find the best density of cells, I created a one-fold serial dilution and counted the cells for each dilution. I found the region where a one-fold dilution would result in a one-fold decrease in cell count. The code can be found on GitLab.
